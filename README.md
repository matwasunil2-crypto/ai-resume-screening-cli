##  About the Project

This project is a simple command-line based application developed as part of the Fundamentals of AI and ML course. The main idea behind this project is to automate the process of resume screening using basic AI concepts.

In real-world scenarios, recruiters often spend a lot of time going through resumes manually. This system tries to reduce that effort by analyzing candidate details like skills and experience, and then ranking them based on job requirements.

The application runs completely in the terminal (CLI), making it lightweight and easy to use.


##  Problem Statement

Manually reviewing resumes is:
- Time-consuming  
- Repetitive  
- Prone to human error  

This project provides a simple automated solution to shortlist candidates efficiently.


##  What This Project Does

- Allows users to **register and login**
- Lets you **add candidate resumes**
- Provides options to **view, edit, and delete resumes**
- Uses AI-based logic to **analyze candidates**
- **Ranks candidates** based on job requirements


## AI Concepts Used

This project uses basic Artificial Intelligence techniques such as:

- Keyword matching (skills comparison)
- Rule-based scoring system
- Simple classification (Selected / Consider / Rejected)
